# Java Calculator

A console-based Java calculator application built using core Java.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Modulus
- Menu-driven execution

## Concepts Used
- Java methods
- switch-case
- do-while loop
- Scanner
- Error handling

## How to Run
1. Compile:
   javac Calculator3.java
2. Run:
   java Calculator3
